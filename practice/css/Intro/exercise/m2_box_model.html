Content-Box

Content-Box sizing will only let the defined width define the size of the content, not the entire size of the element.

The calculation of the content-box element will be Content Width + Padding + Border

Border-Box

Border-Box sizing will let width define the size of the entire element, adjusting the size of the content to fit the determined width.

The calculation of the border-box element will be Content + Padding + Border = Width
